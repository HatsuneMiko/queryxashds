## 如何部署?
## 配置服务
#### 1.在/src/utils/request.js 中配置查询接口基地址
#### 2.在/src/xashds/config.js 里面的ServerList数组中添加服务器对象参数

## 安装依赖
```
npm i --force
```

### 运行
```
npm run serve
```

### 打包,然后把dist文件夹中的内容丢进服务器
```
npm run build
```

### PHP部署
```
直接放到配置有PHP的Apache或者Nginx服务器中
```
